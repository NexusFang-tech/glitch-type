# GLITCH::TYPE — Cyberpunk Text Effects Generator

Real-time animated text effects engine with seven distinct visual modes. Type any text, pick an effect, tweak the parameters, and export production-ready PNGs for stream overlays, thumbnails, social media headers, and portfolio graphics.

## Live Demo

**[nexusfang-tech.github.io/glitch-type](https://nexusfang-tech.github.io/glitch-type)**

## Effects

| Effect | Description |
|--------|-------------|
| **Glitch** | Horizontal slice displacement with RGB channel split and random full-frame tears |
| **Neon Flicker** | Layered glow passes with organic sine-wave driven flicker |
| **Matrix Rain** | Cascading character rain (Latin + Katakana) with floating text window |
| **Scanline** | CRT monitor aesthetic with horizontal line overlay and sweeping scan bar |
| **Hologram** | Vertical wobble with ghost trails and horizontal tear artifacts |
| **Chromatic** | Orbiting RGB channel separation with rotating spread |
| **Typewriter** | Character-by-character reveal with blinking cursor, auto-loops |

## Controls

- **Text Input** — type anything, updates in real time
- **Size** — font size slider (24px–120px)
- **Intensity** — effect strength (1–100)
- **Color** — primary text/effect color picker
- **Background** — canvas background color picker
- **Export PNG** — download the current frame as a PNG
- **Randomize** — shuffle effect, color, size, and intensity
- **Pause/Play** — freeze the animation on a specific frame for export

## Use Cases
